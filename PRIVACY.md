# Privacy

Scrivus is a local-first writing app.

Scrivus does not collect telemetry, analytics, project contents, writing data, or personal information.

Your projects are stored on your computer in the project folders you choose. Scrivus does not upload project files or writing content. If you store a project in a synced folder, export files, share backups, or upload files yourself, that is handled by the service or location you choose.

## Update Checks

When you use Help > Check for Updates, Scrivus contacts GitHub Releases to ask for the latest published Scrivus version. This request does not include your project contents or writing data.

GitHub may receive normal request metadata such as your IP address, user agent, time of request, and network details. GitHub handles that data under GitHub's own privacy practices.

## Backups

Scrivus backups are created locally in the app data folder on your computer. Scrivus does not upload backups.
